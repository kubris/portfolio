Мои услуги

Вёрстка

Верстаю сайты по макету/ образцу/ изображению. Делаю адаптивно/ кроссбраузерно/ валидно. Использую Gulp + SCSS. Проект тестируется на W3C, Google PageSpeed.

Доработка

Добавление/ изменение/ удаление блоков и секций; Мобильная адаптация сайта. Вёрстка и настройка форм обратной связи. Установка сайт на хостинг/ установка SSL.

Администрирование

Установка готовой верстки на CMS WordPress или добавление админки Textalite на HTML-страницу. Установка сайта WordPress на хостинг (подключение базы данных/ привязка SSL).
